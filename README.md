# LOOV — Навигатор по рынку M&A в оптике

Интерактивный навигатор по 85 сделкам и ситуациям в зрении/оптике (2021–2026),
типизированным под три версии продукта LOOV (V1 бренд / V2 гибрид / V3 платформа данных).
Карта рынка и итоги анализа. Источник: внутренний отчёт v2.1 (аудит первоисточников).

Открыть: **index.html** (см. GitHub Pages этого репозитория).
